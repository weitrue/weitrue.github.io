【Go】Etherscan交易详情页面

背景

爬取etherscan交易详情页面的数据。

工具选择

页面数据抓取

直接获取
chromedp

后面发现，有些数据是js动态生成的，于是换成chromedp动态抓取数据。chromedp是一种以更快、更简单的方式来驱动浏览器支持 Go 中的Chrome DevTools 协议，而无需外部依赖。

镜像

chromedp/headless-shell

页面解析

goquery

goquery为Go 语言带来了类似jQuery 语法的工具包。它基于 Go 的net/html 包和 CSS 选择器包cascadia。在语法方面，它尽可能接近 jQuery，尽可能使用相同的函数名称。

因为 net/html 解析器需要 UTF-8 编码，所以 goquery 也是如此。处理非 UTF8 html 页面，可参阅wiki。
